Projete letras do Hinário Adventista usando o OpenLP.

As músicas do hinário estão no formato que o OpenLP reconhece.

Como importar

Após baixar o arquivo usando o botão acima, utilize o menu Arquivo > Importar > Músicas. Você pode selecionar todos os arquivos de uma só vez. A importação pode levar alguns minutos (cerca de 3 a 6 minutos).

Encontrei um problema

Você pode acessar nosso repositório por este link, procurar pela música com problema e editar (você irá localizar o texto do hino muito facilmente!). Você também pode utilizar o botão "Reportar problema" no topo da página.

Não consigo ouvir a música

Infelizmente, para evitar problemas com direitos autorais e tamanho de arquivo, não podemos incluir os arquivos de áudio. No entanto, você pode editar, no próprio OpenLP, suas músicas prediletas e, usando a aba "Áudio Ligado", selecionar seu arquivo de áudio. Ou convidar os jovens de sua igreja para montar uma mini-orquestra ;)

Quem escreveu os arquivos?

Os textos dos hinos foram obtidos no website JesusVoltará.com.br. Após, foi utilizada um pouco de hacking do bem para converter, rapidamente, as músicas para o formato do OpenLP. Você pode conhecer o procedimento neste link.

O projeto é aberto. Sinta-se livre para ajudar :) Projeto iniciado pelo departamento de sonoplastia da Igreja Adventista do Sétimo Dia de Nova Almeida, Serra - ES.
